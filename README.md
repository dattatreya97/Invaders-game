# Invaders-game
Lua game based on Love2d game engine
<br>
# Dependencies (on Linux):
Lua : sudo apt install lua5.1
Love2d : sudo snap install love  # version 11.2+pkg-d332

# Playing the game
love invaders.love/

The game needs more improvement.
